# 人脸识别智能签到系统

基于 Python \+ OpenCV \+ LBPH 人脸识别算法 开发的课堂智能签到系统，适配校园课堂考勤场景，实现人脸采集、模型训练、实时人脸识别、Excel自动签到、重复签到校验等完整功能，界面可视化、操作简单、稳定性强，可作为计算机视觉小型实践、课程设计及实训项目使用。

## ✨ 项目核心功能

- **实时人脸检测**：依托OpenCV级联分类器，精准识别画面中的人脸，绘制可视化人脸框

- **人脸样本采集**：支持摄像头实时采集人脸图像，自定义学生信息命名，自动保存生成人脸数据集

- **LBPH人脸模型训练**：采用经典LBPH人脸识别算法，训练专属本地识别模型，适配日常光照、人脸姿态变化，识别精度高

- **智能自动签到**：人脸识别匹配成功后，自动关联学生信息，写入Excel表格并记录精准签到时间

- **重复签到校验**：自动检索当日签到记录，拦截重复签到操作，弹窗提示用户，杜绝重复考勤

- **智能防误判机制**：设置连续多次识别成功触发签到，有效规避画面抖动、瞬时误识别等问题，提升稳定性

- **双重退出机制**：支持 Q 键手动退出程序，同时配备60秒无操作超时自动退出，运行逻辑更规范

## 💻 运行环境

本项目基于 Python 开发，适配 Windows 系统，需提前配置基础环境并安装全部依赖库。

### 基础环境

- Python 3\.7 / 3\.8 / 3\.9（推荐稳定版本）

- 电脑内置摄像头 / 外接USB摄像头

### 安装依赖库

```Plain Text

pip install opencv-contrib-python
pip install xlrd==1.2.0
pip install xlwt

pip install numpy
pip install Pillow
pip install tk
```

## 🚀 完整使用教程

### 第一步：准备学生信息表格

新建 **人脸签到表\.xls**，按照固定格式创建列表：序号、姓名、学号、班级、签到状态、签到时间，提前录入所有学生基础信息。

### 第二步：采集人脸数据

运行 **人脸采集\.py**，根据提示输入学生姓名、学号，程序自动唤醒摄像头，对准人脸后按 **S键** 采集保存人脸照片，采集的样本自动存入 `data` 文件夹。

### 第三步：训练人脸识别模型

运行 **训练模型\.py**，程序自动读取 `data` 文件夹内的所有人脸样本数据，迭代训练后生成 **train\.yml** 人脸识别模型文件。

### 第四步：启动智能签到

运行主程序 **识别签到\.py**，即可开启智能考勤功能：

- 自动唤醒摄像头，实时检测并识别画面中的人脸

- 连续识别成功10次自动触发签到，自动写入当前时间至Excel表格

- 已完成当日签到的用户，会弹出提示框，禁止重复签到

- 按 **q键** 可手动关闭程序，无操作60秒自动退出
